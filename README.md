IPL Performance Analysis 🏏

This project analyzes Indian Premier League (IPL) match and player data using Python to uncover trends and insights.

Tools & Technologies

Python

Pandas

Matplotlib / Seaborn

Dataset

IPL dataset containing match results, player stats, and team performances

Key Analysis

Team performance comparison across seasons

Player statistics: top scorers, top wicket-takers

Head-to-head team analysis

Run rate and win/loss trend exploration

Visualizations

Bar chart (team wins, top players)

Histogram (runs, wickets distribution)

Line chart (team performance over seasons)

Insights

Certain teams dominate specific seasons

Players excel in different skill areas (batting vs bowling)

Run rates vary significantly across matches and venues

How to Run
pip install pandas matplotlib seaborn
python ipl_analysis.py
